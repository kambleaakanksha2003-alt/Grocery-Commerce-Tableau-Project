# Grocery-Commerce-Tableau-Project

## 🚀 Project Overview

The Hyperlocal Grocery Commerce Analytics Project is an end-to-end Business Intelligence solution developed using Tableau.

This project analyzes order-level transactional data of a hyperlocal grocery delivery business to uncover insights across revenue, customers, products, and delivery operations.

The objective is to transform raw data into interactive dashboards that support strategic decision-making
